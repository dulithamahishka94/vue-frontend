 1. Clone this repository
 2. Go to the cloned folder and run `npm install`
 3. Create a copy of the .env.example file as .env
 4. Update VUE_APP_API_ENDPOINT with backend virtual host URL in .env file
 5. Run `npm run serve`
 6. Done
